PROJECT: Korion Official Website — Infrastructure-Level Web3 Product UI Design

OBJECTIVE:
Design a modern, premium, infrastructure-grade website UI for Korion, a revolutionary offline crypto payment system that enables secure smartphone-to-smartphone digital asset transactions without internet connectivity.

The design quality must match or exceed the visual standards of arbitrum.io, layerzero.network, and starknet.io.

The design must communicate trust, advanced infrastructure, security, and global financial scale.

---

GLOBAL DESIGN STYLE:

Theme:

* Dark, futuristic, infrastructure-grade interface
* Background: deep black (#050505 or similar)
* Accent colors: electric blue, deep purple, subtle gradient glow
* Clean, minimal, modern

Visual tone:

* premium
* futuristic
* secure
* scalable infrastructure

Use visual techniques such as:

* glassmorphism
* soft glow effects
* subtle gradients
* modern spacing
* clean typography hierarchy

Avoid:

* cartoonish elements
* overly colorful designs
* cluttered layouts

---

TYPOGRAPHY SYSTEM:

Primary font:

* Inter, Satoshi, or Space Grotesk

Font hierarchy:

Hero headline:

* 64–80px
* bold
* strong presence

Section headers:

* 36–48px
* semi-bold

Body text:

* 16–18px
* regular
* high readability

Use proper spacing and hierarchy.

---

GRID AND LAYOUT SYSTEM:

Use 12-column grid layout.

Max width:

* 1200–1400px container

Spacing system:

* 8px spacing scale
* consistent vertical rhythm

Sections must have strong spacing and visual separation.
